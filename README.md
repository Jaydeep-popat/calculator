# Glass Calculator - Next.js

A beautiful glass morphism calculator built with Next.js, featuring:

- **Basic Mode**: Standard calculator operations
- **Scientific Mode**: Advanced math functions (sin, cos, tan, ln, log, factorial, etc.)
- **Currency Converter**: Multi-currency conversion with customizable rates

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the calculator.

## Features

- Glass morphism UI design
- Keyboard shortcuts support
- Responsive design
- Three modes: Basic, Scientific, Currency
- Memory functions
- Degree/Radian toggle for trigonometric functions
- Customizable currency exchange rates

## Keyboard Shortcuts

- **0-9**: Enter numbers
- **+, -, *, /**: Basic operations
- **Enter** or **=**: Calculate result
- **Escape**: Clear all
- **Backspace**: Delete last character
- **%**: Percentage
